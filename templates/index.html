Modules OSINT avancés (roadmap)

  • LeakCheck : vérification d'emails dans des bases de fuites (API payante).
  • ExifTool : extraction de métadonnées d'images (local ou upload).
  • Sherlock / Skiptracer : recherche de profils & empreinte numérique d'un pseudo.
  • Sentinel / satellites : vision géospatiale (imagerie) pour certaines enquêtes.

L'architecture actuelle (API FastAPI + front en JS) est prête pour intégrer ces modules additionnels, chacun sous forme d'endpoint spécialisé.

Installation Chrome / Edge / Brave
  1. Lancez lancer_web.bat (serveur sur 127.0.0.1:8000)
  2. Ouvrez chrome://extensions ou edge://extensions
  3. Activez le Mode développeur
  4. Charger l'extension non empaquetée → dossier extension/ du projet

Chemin : …/phishing_guardian/extension

Fonctionnalités
  • 🎣 Analyse phishing de l'URL de la page active
  • ⚡ Playbook OSINT rapide sur email, domaine, IP ou pseudo
  • 🔒 Scores Attack Surface & Privacy dans le popup
  • 📋 Menu contextuel : clic droit → OSINT sur la sélection
  • 🛡 Ouverture directe du dashboard Phishing Guardian
